# gbackup

CLI backup tool

#### Usage :

```
gbackup Directory
```

eg : ```gbackup Downloads```

#### Installation :
```
git clone https://github.com/polarspetroll/gbackup.git
cd gbackup
go build -o gbackup main.go
sudo mv gbackup /usr/local/bin
```

- Make sure to add the final destination (main.go:17)
